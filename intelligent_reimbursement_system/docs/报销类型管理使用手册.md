# 报销类型管理 使用手册

本页面用于创建和管理报销类型，管理员可以在这里定义不同业务场景下的报销表单结构、计算规则和导出格式。

---

## 一、AI 智能填写（推荐）

页面右下角有一个 **小智助手** 悬浮按钮，可以用自然语言描述你想要的报销类型，AI 会自动生成完整配置并填入表单，无需手动逐项填写。

**使用步骤：**

1. 点击右下角 🤖 图标打开小智助手
2. 输入描述，例如：
   - `帮我创建一个差旅报销类型，包含出发地、目的地、交通费、住宿费字段`
   - `我需要一个采购报销，有商品名称、单价、数量，总价自动计算`
3. AI 返回配置预览后，点击卡片上的 **「一键填入表单 →」** 按钮
4. 表单自动填充，确认内容无误后点击 **保存**

> 如果你已经在报销类型管理页面，点击按钮后表单会直接填充，无需跳转。

## 二、手动新建报销类型

### 2.1 基本信息

| 字段       | 说明                                               | 示例                   |
| ---------- | -------------------------------------------------- | ---------------------- |
| 类型标识符 | 英文唯一标识，只能包含字母和下划线，提交后不可重复 | `travel`、`purchase`   |
| 类型名称   | 中文展示名称，不可重复                             | `差旅报销`、`采购报销` |
| 备注       | 选填，对该类型的补充说明                           | `适用于出差场景`       |
| 计算公式   | 用于自动计算报销总价，变量名对应字段标识符         | `unitPrice * quantity` |
| 超额标准   | 报销金额超过该值时标记为超额，不填则不限制         | `5000`                 |
| 状态       | 启用后该类型才会出现在报销申请的下拉列表中         | 启用 / 禁用            |

**计算公式说明：**

公式中的变量名必须与下方字段配置中「参与计算」开关打开的字段的**标识符**完全一致。

```
示例：字段有 unitPrice（单价）和 quantity（数量）
公式写：unitPrice * quantity
系统会在提交报销时自动计算总价
```

---

### 2.2 字段配置

字段是用户填写报销申请时看到的表单项。点击 **「添加字段」** 按钮新增一行。

| 列名       | 说明                                                 |
| ---------- | ---------------------------------------------------- |
| 字段标识符 | 英文，用于公式引用和数据存储，如 `unitPrice`         |
| 字段名称   | 中文展示名，用户填写时看到的标签，如 `单价`          |
| 字段类型   | 见下方类型说明                                       |
| 必填       | 开启后用户提交时该字段不能为空                       |
| 参与计算   | 开启后该字段的值会被代入计算公式，类型必须为**数字** |
| 排序       | 数字越小越靠前，控制字段在表单中的显示顺序           |

**字段类型说明：**

- `文本` — 单行文字输入
- `数字` — 数值输入，支持小数，可参与公式计算
- `日期` — 日期选择器
- `下拉` — 单选下拉，需要配置选项（见下方）
- `多行文本` — 多行文字输入，适合备注类字段

**下拉字段选项配置：**

字段类型选择「下拉」后，点击该行左侧的展开箭头，可以添加选项。每个选项需要填写：

- **选项名称**：用户看到的文字，如 `飞机`
- **选项值**：存储到数据库的值，如 `airplane`

---

### 2.3 导出字段配置

导出字段决定了点击「导出 Excel」时，表格中包含哪些列以及如何计算。

点击 **「添加导出字段」** 新增一行，支持拖拽 ☰ 图标调整列的顺序。

| 列名         | 说明                                           |
| ------------ | ---------------------------------------------- |
| 字段标识符   | 对应表单字段的标识符，填写后会自动取该字段的值 |
| 导出列名     | Excel 中显示的列标题                           |
| 参与计算字段 | 如果该列需要计算，选择参与计算的数字字段       |
| 计算公式     | 该列的计算逻辑，变量名对应上方选择的字段标识符 |

**示例：** 导出一列「小计」，值为单价 × 数量

```
字段标识符：subtotal
导出列名：小计
参与计算字段：unitPrice、quantity
计算公式：unitPrice * quantity
```

---

### 2.4 保存

填写完成后点击右下角 **「保存」** 按钮。系统会校验：

- 类型标识符和名称不能与已有类型重复
- 所有字段必须填写标识符和名称
- 下拉类型字段至少需要一个选项
- 所有导出字段必须填写标识符和列名

## 三、查看已有报销类型

页面下方展示所有已创建的报销类型列表，包含标识符、名称、字段数、导出字段数。

点击 **「详情」** 可以查看该类型的完整字段配置和导出字段配置。

---

## 四、修改报销类型

点击列表中的 **「修改」** 按钮，弹出编辑弹窗。

编辑弹窗中的字段配置支持**拖拽排序**，拖动每行左侧的 ☰ 图标即可调整字段顺序，排序值会自动更新。

修改完成后点击 **「保存修改」**。

---

## 五、删除报销类型

点击列表中的 **「删除」** 按钮，确认后删除。

> 注意：删除后已使用该类型的报销记录不受影响，但新的报销申请将无法选择该类型。

## 六、常见问题

**Q：计算公式写了但总价没有自动计算？**

检查以下几点：

1. 公式中的变量名是否与字段标识符完全一致（区分大小写）
2. 参与计算的字段类型是否为「数字」
3. 参与计算的字段「参与计算」开关是否已打开

**Q：下拉字段的选项值填什么？**

选项值是存储到数据库的实际值，建议使用英文或数字，选项名称才是用户看到的中文。两者可以相同，但建议区分。

**Q：导出字段和表单字段有什么区别？**

表单字段是用户填写报销申请时看到的输入项；导出字段是导出 Excel 时的列定义，可以包含计算列（如小计、合计），也可以只是表单字段的子集。
